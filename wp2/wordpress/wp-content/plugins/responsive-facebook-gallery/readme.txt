Responsive facebook gallery, just plug and play

== Description ==

Responsive Facebook Gallery helps you to integrate facebook gallery easily to your website in few easy steps.

It is compatible with all major browsers IE8+, firefox, chrome, safari, opera and android browsers

Once its activated enter the shortcode to the page where you want to display gallery.

An online demo can be found at http://pixsols.com/test/wordpress/responsive-facebook-gallery/.

== Installation ==

1. Unzip and upload the `facebookgallery` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
4. Use the shortcode `[fbgallery url='FACEBOOK GALLERY COMPLETE URL' column='{1,2,3,4}']` in your posts or pages.

== Frequently Asked Questions ==

= What does the column parameter do =
You can change the number of images in a row by changing numbers.
